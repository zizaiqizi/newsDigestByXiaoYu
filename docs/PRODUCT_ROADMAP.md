## 🔄 持续迭代机制

### 每周循环
```
周一: 评审research-agent竞品报告
周二: 确定本周优化优先级
周三-五: 开发新功能
周六: 测试 + 代码审查
周日: 发布 + 数据复盘
```

### 数据驱动
- 每日监控: 转化率/留存率/付费率
- 每周分析: 用户行为路径
- 每月复盘: ROI和LTV变化
