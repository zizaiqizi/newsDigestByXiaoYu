## 📈 Part 4: 财经观察 - 新增内容

### 每日分析维度

1. **A股市场**
   - 隔夜美股对A股影响
   - 当日热点板块分析
   - 龙虎榜资金流向
   - 政策/消息面解读

2. **加密货币市场**
   - BTC/ETH隔夜行情
   - 市场恐慌/贪婪指数
   - 山寨币异动监测
   - 合约资金费率

3. **技术指标扫描**
   - 双均线交叉信号
   - MACD背离/金叉死叉
   - 成交量异常放大
   - 支撑位/阻力位

4. **当日投资建议**
   - 明确标的（股票代码/币种）
   - 建议方向（做多/做空/观望）
   - 入场价位区间
   - 止损价位（严格<5%）
   - 目标价位

## ⚠️ 风险提示模板

每份投资分析必须包含：

```
⚠️ 风险提示
- 本分析仅供参考，不构成投资建议
- 投资有风险，入市需谨慎
- 请根据个人风险承受能力决策
- 建议单次投资不超过总资金的20%
- 严格止损，单次亏损控制在5%以内
```

## 💰 资金管理

**账户信息**:
- 初始本金: ¥10,000
- 核算周期: 每月1号
- 分红比例: 盈利部分的20%

**风险控制**:
- 单次交易最大亏损: 5%
- 总账户最大回撤: 20%
- 杠杆使用: 不超过2倍
